=== COMANDOS QUE EU JA SEI ===

NAVEGACAO:

pwd	- mostra onde estou
ls	- lista aquivos
ls -la  - lista com detalhes
cd	- entra em pasta
cd ..   - volta uma pasta
cd ~    - vai para a home


CRIAR E DELETAR

mkdir	- cria pasta
touch	- cria arquivo vazio
rm	- deleta aquivo
rm -rf  - deleta pasta inteira

LEITURA

cat	- mostra conteudo
less    - abre para leitura
grep    - procura texto

=== BUSCA E FILTRO ===

grep "texto" arquivo          - procura texto dentro do arquivo
grep "texto" arquivo -r       - procura em todos os arquivos da pasta
find /pasta -name "*.sh"      - encontra arquivos pelo nome
tail -f arquivo               - monitora arquivo ao vivo
wc -l                         - conta linha
cmd1 | cmd2                   - pipe: passa resultado de um para outro


=== EXEMPLOS REAIS ===

grep "error" /var/log/nginx/error.log  - ver erros do Nginx
tail -f /var/log/nginx/access.log      - monitorar acessos ao vivo
find ~ -name "*.sh" | wc -l            - contar scripts


=== PROCESSOS ===

ps aux               - lista todos os processos rodando
ps aux | grep nginx  - filtra processo específico
top                  - monitor em tempo real (q para sair)
kill PID             - para um processo pelo número
kill -9 PID          - força parada imediata

=== SYSTEMCTL - CONTROLAR SERVIÇOS ===

systemctl status nginx   - ver status do serviço
systemctl start nginx    - iniciar
systemctl stop nginx     - parar
systemctl restart nginx  - reiniciar
systemctl enable nginx   - iniciar automaticamente no boot

=== CRON - AGENDAMENTO ===

crontab -e   - editar agendamentos
crontab -l   - listar agendamentos

FORMATO: minuto hora dia mês diaSemana comando
0 8 * * *    - todo dia às 8am
*/5 * * * *  - a cada 5 minutos
0 * * * *    - todo hora
